# 📊 Library Survey Automation System

Google Apps Script와 Gemini API를 활용한  
**범용 만족도 조사 분석 및 자동 보고서 생성 시스템**입니다.

네이버 폼 등에서 수집한 설문조사 Excel 파일을 업로드하면  
설문 문항 구조를 자동으로 분석하고, 사용자가 분석 유형을 확인·수정한 뒤  
통계 분석부터 AI 기반 주관식 분석, 총평 및 향후계획 작성, Excel 결과보고서 생성까지 자동으로 처리합니다.

---

## 📌 프로젝트 소개

도서관 및 공공기관에서는 다양한 사업과 프로그램 운영 후 만족도 조사를 실시합니다.

기존 만족도 조사 업무는 설문마다 문항 구조가 달라 다음과 같은 작업을 반복해야 합니다.

- Excel 응답 데이터 정리
- 개인정보 및 분석 제외 문항 구분
- 단일응답·복수응답 문항 분석
- 5점 척도 만족도 계산
- 주관식 의견 정리
- 결과보고서 작성
- 차트 및 대시보드 생성

본 프로젝트는 이러한 반복 업무를 줄이기 위해 개발한  
**범용 만족도 조사 자동화 시스템**입니다.

특정 설문 양식에 종속되지 않고 Excel 파일의 문항 구조를 분석하여  
다양한 만족도 조사에 활용할 수 있도록 설계했습니다.

---

## ✨ 주요 기능

### 1. 조사 기본정보 관리

조사명, 조사목적, 조사기간, 조사대상, 조사방법, 담당부서 등의 정보를 입력하고 관리합니다.

---

### 2. Excel 설문 데이터 업로드

네이버 폼 등에서 내려받은 Excel 설문 응답 파일을 업로드할 수 있습니다.

업로드된 Excel 파일에서 다음 정보를 자동으로 확인합니다.

- 원본 문항
- 컬럼 번호
- 예시 응답
- 원본 시트명
- 응답 데이터

---

### 3. 설문 문항 자동 분석 및 매핑

업로드된 설문 문항과 예시 응답을 분석하여 적절한 분석 유형을 자동 추천합니다.

지원하는 분석 유형은 다음과 같습니다.

| 분석 유형 | 설명 |
|---|---|
| EXCLUDE | 분석 제외 |
| RESPONDENT | 응답자 특성 |
| PERSONAL_INFO | 개인정보 |
| SINGLE | 단일응답 |
| MULTIPLE | 복수응답 |
| SCALE | 5점 척도 만족도 |
| RECOMMENDATION | 재이용·추천 의향 |
| TEXT | 주관식 의견 |

자동 추천된 분석 유형은 사용자가 직접 확인하고 수정할 수 있습니다.

---

### 4. 범용 원자료 생성

최종 선택된 문항 매핑 정보를 기준으로 분석용 범용 원자료를 생성합니다.

설문마다 문항 개수와 구조가 달라도 동적으로 처리할 수 있도록 구성했습니다.

---

### 5. 자동 통계 분석

문항 유형에 따라 자동으로 통계를 계산합니다.

주요 분석 항목은 다음과 같습니다.

- 응답자 수
- 단일응답 빈도 분석
- 복수응답 분석
- 선택 건수
- 응답자 기준 선택률
- 전체 선택 기준 비율
- 5점 척도 평균
- 100점 환산 점수
- 긍정·중립·부정 응답률
- 만족도 문항 순위
- 전체 평균 대비 편차
- 재이용 및 추천 의향 분석
- 주관식 의견 추출

---

### 6. 자동 결과보고서 생성

분석 결과를 기반으로 Google Spreadsheet에 결과보고서를 자동 생성합니다.

생성되는 시트는 다음과 같습니다.

| 시트 | 내용 |
|---|---|
| 01_조사개요 | 조사 기본정보 |
| 02_대시보드 | 핵심 통계 및 차트 |
| 03_응답자특성 | 응답자 특성 분석 |
| 04_복수응답분석 | 복수응답 결과 |
| 05_만족도분석 | 5점 척도 및 만족도 결과 |
| 06_주관식분석 | AI 기반 주관식 의견 분류 |
| 07_AI총평 | Gemini 기반 결과 총평 |
| 08_향후계획 | 조사 결과 기반 향후계획 |
| 09_범용원자료 | 분석용 원자료 |

---

### 7. Gemini AI 기반 주관식 분석

Gemini API를 활용하여 주관식 의견을 의미가 유사한 범주로 자동 분류합니다.

AI 분석 과정에서 다음과 같은 검증 로직을 적용했습니다.

- 실제 원자료에 존재하는 응답만 분석
- AI가 반환한 응답 번호 검증
- 범주별 건수를 Apps Script에서 재계산
- 연락처 및 이메일 비식별 처리
- 조사 결과에 존재하지 않는 정보 생성 제한
- 담당자 최종 검토 안내 표시

---

### 8. AI 총평 및 향후계획 생성

통계 결과와 주관식 분석 결과를 Gemini에 전달하여 공공기관 결과보고서 형식의 초안을 생성합니다.

AI가 임의의 정책, 예산, 일정 등을 생성하지 않도록 프롬프트를 제한하고  
실제 설문 분석 결과를 기반으로 문안을 작성하도록 설계했습니다.

---

### 9. 웹 대시보드

Google Apps Script Web App을 활용하여 별도의 프로그램 설치 없이 브라우저에서 시스템을 사용할 수 있습니다.

업무 처리 순서는 다음과 같습니다.

조사정보 입력
    ↓
Excel 파일 선택
    ↓
문항 구조 자동 분석
    ↓
분석 유형 확인 및 수정
    ↓
범용 원자료 생성
    ↓
통계 보고서 생성
    ↓
Gemini AI 분석
    ↓
결과 확인
    ↓
Excel 보고서 저장

각 단계의 완료 상태에 따라 다음 기능 버튼이 순차적으로 활성화되도록 구성했습니다.

---
### 10. Excel 결과보고서 다운로드

최종 생성된 결과보고서를 Excel(.xlsx) 파일로 변환하여 저장할 수 있습니다.

내부 관리용 설정 및 문항 매핑 시트는 제외하고 실제 결과보고서 시트만 내보냅니다.

### 🎯 프로젝트 개발 목적

도서관 인턴 업무 중 만족도 조사 결과를 정리하고 보고서를 작성하는 과정에서 반복적인 데이터 처리 업무를 자동화하기 위해 개발했습니다.

초기에는 특정 만족도 조사 양식을 처리하는 시스템으로 개발했으나, 설문마다 문항 구조가 다르다는 문제를 해결하기 위해 문항 자동 분석과 사용자 매핑 기능을 추가하여 범용 시스템으로 확장했습니다.

또한 단순 통계 자동화를 넘어 Gemini API를 활용한 주관식 의견 분석, 결과 총평 및 향후계획 작성 기능을 구현했습니다.
